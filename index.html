A. Instalación servidor Web

Para la instalación del servidor Web elegí la opcion Apache debido a su facilidad de instalación y a que ya lo había manejado antes. Durante el proceso de instalación, primero tuve que instalar el paquete de apache (apache2) y verificar que estuviera activo.

Los comandos utilizados fueron los siguientes:
$ sudo apt update ###Para actualizar los paquetes
$ sudo apt install apache2 ###Instalar el paquete de apache
$ systemctl status apache2 ###Verificar que el servicio esté activo
$ curl -4 icanhazip.com ###Devuelve la IP pública donde está el servidor.

Luego de esto se debía cambiar el puerto por defecto del servidor, el cual debía ser 8088 y se cambiaba modificando el archivo /etc/apache2/ports.conf, añadiendo el puerto solicitado y además también se debe modificar el archivo /etc/apache2/sites-enabled/000-default.conf y por último se reiniciaba el servidor apache.
